Charades Usage Tracker Test

Manual Testing Instructions

Open the Charades app at http://localhost:8080 and follow these steps to test the usage tracking feature:

  1. Click on different category buttons (Film, TV show, etc.) multiple times
  2. Notice that the same items should not appear repeatedly if usage tracking is working
  3. Open the settings panel (gear icon in top-right)
  4. Find and click the "Reset Used Content" button
  5. Verify that previously used items can now be selected again
